Welcome to the profiles prototype gallery

The profiles prototype gallery is a central place to store prototypes, including both code-based and Figma prototypes.

Using the Profiles Prototype gallery

You can navigate to the prototypes of a given sector by clicking on the sector name in the navigation header above. You can then filter the prototypes to see viewer specific or editor speific prototypes.

Accessing mobile prototypes

Code based prototypes natively work on all devices types, including mobile phones, tablets and desktops/laptops.

Figma prototypes aren't responsive, and as such, multiple files need to be created for each viewport (mobile, tablet, etc.). You'll find the device type prepended to the link for each Figma prototype.
