Register for the new NHS Profile Manager

The new NHS Profile Manager combines the current NHS website profile editor and the Directory of Services (DoS) Profile Updater.

The new system will update both the NHS website and the DoS. You will be able to save time with 1 system, to meet your contractual obligations for updating profiles.

Secure access with NHSmail

Register for the NHS Profile Manager with your personal NHSmail address only. You will not be able to log in with shared NHSmail addresses or other types of email.
